# Ideas para próximas iteraciones

## Preview del ebook en la landing

**Idea:** Mostrar 4-5 páginas del ebook en un carrusel o flip-book interactivo, para que el usuario vea qué hay adentro antes de comprar.

**Por qué:** El 75% de los visitantes abandona antes de ver el precio. Parte del problema es que no tienen claro qué están comprando. Ver páginas reales del ebook genera confianza y aumenta la intención de compra.

**Dónde en la landing:** Entre la sección de Módulos y la de Precio — cuando ya saben qué hay adentro y están a punto de decidir.

**Implementación sugerida:**
- Librerías JS: Turn.js o StPageFlip para efecto de hojear páginas real
- Exportar las páginas elegidas como imágenes desde el PDF (se puede automatizar con Python + pdfplumber)

**Páginas candidatas a mostrar:**
- Portada — refuerza el producto
- Psicología del color — página visual, muestra profundidad de contenido
- Una página de ejercicio práctico — demuestra que no es solo teoría
- Una página del módulo de Canva — muestra lo accesible y práctico
- Página de cierre con el mensaje de Lucía — conexión emocional

**Pendiente antes de implementar:**
- Definir con Lucía qué páginas quiere mostrar (puede haber contenido que prefiera reservar para compradores)
